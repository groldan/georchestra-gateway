= GeoFence Integration

=== GeoFence resource access manager 

[source,mermaid]
----
sequenceDiagram
  actor Gateway
  participant WMS
  participant SecureCatalog
  participant CatalogImpl
  participant GeofenceAccessManager
  participant GeoFenceConfiguration
  
  Gateway->>+WMS: ?request=GetMap&layer=name&...
  note right of Gateway: sec-username: test@gdi.de, sec-roles: ROLE_GDI_PLANER_EXTERNAL
  WMS->>+SecureCatalog: getLayer(name)
  SecureCatalog->>+CatalogImpl: getLayer(name)
  CatalogImpl-->>-SecureCatalog: LayerInfo
  SecureCatalog->>SecureCatalog: user() Authentication[roles: ROLE_GDI_PLANER_EXTERNAL]
  
  SecureCatalog->>+GeofenceAccessManager: getAccessLimits(Authentication, LayerInfo)
  GeofenceAccessManager->>+GeoFenceConfiguration: filter using roles?
  GeoFenceConfiguration-->>-GeofenceAccessManager: true
  GeofenceAccessManager->>GeofenceAccessManager: build RuleFilter from first valid role from GeoFenceConfiguration
  
  GeofenceAccessManager->>+RuleReaderService: getAccessInfo(RuleFilter)
  RuleReaderService->>+RuleDAO: find rules (user: Any, role: RuleFilter's role)
  RuleDAO-->>-RuleReaderService: List<Rule>
  RuleReaderService->>RuleReaderService: Create AccessInfo with resolved Rules
  RuleReaderService-->>-GeofenceAccessManager: AccessInfo
  
  GeofenceAccessManager-->>-SecureCatalog: DataAccessLimits
  
  SecureCatalog->>SecureCatalog: checkAccess(LayerInfo, DataAccessLimits)
  SecureCatalog-->>-WMS: SecuredLayerInfo
  WMS->WMS: render(layer)
  WMS-->>-Gateway: image
----

Use case: Authbot scheduled reconcile

[source,mermaid]
----
sequenceDiagram
  participant bot as AuthBot
  participant eam as FibreEAM
  participant gs as GeoServer
  participant geofence as Geofence
  
  loop Every scheduled period
    bot->>eam: get all rules by company
    eam-->>bot: {companyId, rules[]}+
    bot->>bot: build Gefence rules
    bot->>gs: get all Roles
    gs-->>bot: [ROLE_1,..., GROUP_<companyId>, ...]
    loop for each missing company->role name
      bot->>gs: create company role GROUP_<companyId>
    end
    loop for each company
      bot->>geofence: get all rules for GROUP_<companyId>
      geofence-->>bot: Rule[]
      bot->>bot: compute mismatched rules
      loop for each mismatched rule
        bot->>geofence: create or update rule
      end
    end
  end
----
